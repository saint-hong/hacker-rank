# Problem 1/30

##"Data Types"
  - 3개의 데이터 타입을 입력받아서 다음과 같이 출력하시오.

## input
  - i = 4
  - d = 4.0
  - s = "HackerRank"
## output
  - 16 (i 와 다른 변수의 합)
  - 8.0 (d 와 다른 변수의 합)
  - HackerRank is the best place to learn and practice coding! (s 와 다른 변수의 합)

## solution !!
i = 4
d = 4.0
s = "HackerRank"

data1 = 16
data2 = 4.0
data3 = "is the best place to learn and practice coding!"

print(i+data1)
print(d+data2)
print(s+data3)
